# 🕵️‍♂️ 소머즈 에이전트 (SomersAgent)
> **출처가 명확한 전문지식 문서 검색 및 AI 학습 자동화 파이프라인**

본 프로젝트는 고신뢰도 문서(논문, 정부/공공기관 보고서, 국제 표준 규격 등)를 수집하여 신뢰도를 자체 평가하고, AI 학습에 적합한 지식 청크(RAG/Fine-Tuning 용 Q&A 페어)로 변환 및 축적하여 깃허브 저장소로 자동 동기화하는 **베테랑 소머즈 에이전트**입니다.

---

## 🚀 주요 기능
1. **대화식 조건 사전 설정**: 학습을 시작하기 전, 학습 주제, 허용할 최신성 연도 범위, 상세 검토 조건 등을 직접 대화형으로 설정할 수 있습니다.
2. **7대 신뢰도 평가 엔진**: 수집된 각 문서에 대해 출처 신뢰도, 최신성, 전문성, 원문성, 실무성, 데이터 밀도, 객관성을 계량적으로 점수화하여 종합 별점을 부여합니다.
3. **고효율 지식 청크 변환**: 수집된 텍스트 데이터를 AI가 학습하기 용이하도록 정밀한 Q&A 세트로 가공하여 로컬 지식베이스에 누적 적재합니다.
4. **다차원 보고서 자동 작성**: 한글 시각화 평점 차트(`png`), 상세 분석 보고서(`md`), 결재용 고품질 Word 보고서(`docx`)를 일괄 생성합니다.
5. **깃허브 자동 원격 백업**: 학습 완료 후 생성된 모든 지식과 산출물을 연결된 깃허브 저장소로 즉시 강제 푸시(Force Push)하여 동기화합니다.

---

## 🛠️ 설치 및 구동 방법

### 1. 환경 및 패키지 설치
로컬 가상환경 활성화 상태에서 아래 라이브러리 설치가 필요합니다:
```bash
uv pip install pyyaml python-docx matplotlib koreanize-matplotlib setuptools
```

### 2. 에이전트 가동 (대화식 모드)
터미널에서 아래 스크립트를 실행하면 학습 조건을 물어보는 대화형 프롬프트가 실행됩니다:
```bash
.\.venv\Scripts\python.exe run_search.py
```
* **대화형 질문 순서**:
  1. 오늘 학습할 전문분야 주제 입력 (예: `식품공장 위생배관`)
  2. 허용할 문서의 최신성 기준 연도 입력 (예: `3`)
  3. 학습 시 반영할 추가 상세 요청사항 입력 (예: `HACCP 기준 및 STS316L 위주`)
  4. 깃허브 저장소로 바로 동기화할까요? (`y` 입력 시 자동 Push 구동)

## 📂 저장소 파일 구조
* 📂 **`knowledge_base.json`**: 매일 학습한 정보와 가공된 Q&A 지식 청크가 누적 적재되는 JSON 데이터베이스
* 📂 **`somers_rules.yaml`**: 에이전트 페르소나 및 7대 신뢰도 검증 평가 기준 설정서
* 📂 **`somers_agent.py`**: 쿼리 플래너, 신뢰도 평가 엔진, Git 동기화 매니저가 통합된 에이전트 코어 코드
* 📂 **`run_search.py`**: CLI 구동 및 대화형 인터페이스 제공 엔트리 파일
* 📊 **`신뢰도_분포.png`**: 수집된 문서들의 5대 지표 평균 점수를 보여주는 시각화 차트
* 📄 **`학습결과_보고서.md` / `.docx`**: 지식베이스 요약 및 실무 적용 포인트를 정리한 마크다운 및 워드 보고서
