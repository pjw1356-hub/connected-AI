# [학술논문] 딥러닝 기반 3D 비전 인식을 활용한 식품 공정 무작위 빈피킹(Random Bin-Picking) 시스템 개발

* **출처**: 한국농업기계학회지 (Journal of Biosystems Engineering)
* **저자/기관**: 이정훈, 최성우 (서울대학교 푸드테크학과)
* **발행연도**: 2025년 3월 (Volume 50, Issue 1)
* **주요 요약**:
  * **연구 배경**: 컨베이어 벨트나 원자재 상자(Bin)에 무작위로 쌓여 있는 농수산물은 고유의 기하학적 형태가 모두 다르고 그림자나 중첩으로 인해 기존 2D 카메라 기반 비전 센서로는 정확한 위치 파악이 어려움.
  * **핵심 기술**:
    * **Mask R-CNN 기반 인스턴스 세그멘테이션**: 학습된 인공지능 모델을 통해 겹쳐져 있는 각 농산물(사과, 참외 등)의 경계를 개별 분할 인식함.
    * **3D 점구름(Point Cloud) 매핑**: 깊이 카메라(Depth Camera)를 사용하여 대상체의 공간적 자세(Pose, x/y/z 및 롤/피치/요) 데이터를 획득함.
    * **충돌 없는 경로 계획(Collision-free Path Planning)**: 빈(Bin)의 벽면이나 다른 물체와의 충돌을 피하며 그리퍼가 최적 각도로 대상체에 접근하도록 가이드라인을 제시하는 알고리즘.
  * **실험 결과**: 무작위 적재된 과채류를 피킹하는 시간은 개당 평균 0.8초 이내이며, 자세 오류로 인한 파지 실패율을 기존 대비 85% 이상 개선함.
  * **식품공장 적용점**: 원자재 세척 라인 투입구, 원재료 선별 공정의 입구 단 자동화에 적합함.
